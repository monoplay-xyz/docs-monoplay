# Welcome to MonoPlay

MonoPlay is a decentralized game distribution platform built on the Monolythium blockchain. We're reimagining how games are built, distributed, and played ‚Äî putting ownership and rewards back in the hands of players, developers, and the community.

## The MonoPlay Ecosystem

MonoPlay consists of three interconnected products:

### üéÆ MonoPlay Platform

The complete game store and launcher experience. Browse, purchase, and play games with blockchain-powered licensing and wallet-based authentication. No accounts to manage, no DRM hassles ‚Äî just games.

- **Storefront**: Discover games across all genres
- **Desktop Launcher**: Manage your library and downloads
- **On-Chain Licensing**: Provably own your games forever
- **LYTH Payments**: Fast, low-cost transactions

### üõ†Ô∏è MonoPlay Forge

- **AI Game Builder**: Describe your game, get a playable prototype
- **Bevy Engine Integration**: Modern Rust-based game engine
- **SDK Library**: Physics, networking, UI, audio modules
- **One-Click Publishing**: Deploy directly to the Platform

[Learn more about Forge ‚Üí](/forge/overview)

### üåê MonoPlay GRID

A decentralized content delivery network (CDN) powered by community-run nodes. Games are distributed via torrent protocol, ensuring fast downloads and censorship resistance. Run a GRID node and earn LYTH rewards for seeding games.

- **Torrent Distribution**: Faster downloads, lower infrastructure costs
- **Seeder Rewards**: Earn LYTH for contributing bandwidth
- **Redundancy**: Games stay available even if the platform goes offline
- **Low Latency**: Download from nearby nodes

[Learn more about GRID ‚Üí](/grid/overview)

## Why MonoPlay?

### For Players
- **True Ownership**: Your licenses live on-chain, not locked in a corporate database
- **No DRM**: Play offline anytime with cached license tokens
- **Lower Prices**: Decentralized distribution = lower costs for developers = better prices
- **Privacy**: Wallet-based authentication, no email required

### For Developers
- **Fair Revenue Share**: Keep more of what you earn
- **AI-Powered Tools**: Build faster with MonoPlay Forge
- **Global Distribution**: Instant worldwide reach via GRID
- **Security Scanning**: Automated malware and exploit detection

### For Node Operators
- **Passive Income**: Earn LYTH by seeding popular games
- **Low Requirements**: Run on consumer hardware
- **Transparent Rewards**: All payouts recorded on-chain
